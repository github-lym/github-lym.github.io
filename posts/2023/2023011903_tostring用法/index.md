# ToString()用法

突然找到我完全不知道的用法。  
<!--more-->
|  參數	|含意|Input|Output|
|  ----  | ----  | ----  | ----  |
|C|貨幣|2.5.ToString("C")|NT2.5|
|D|十進制數|25.ToString("D5")|00025|
|E|科學型|25000.ToString("E")|2.500000E+005|
|F|固定點|25.ToString("F2")|25.00|
|G|常規|2.5.ToString("G")|2.5|
|N|數字|2500000.ToString("N")|2,500,000.00|
|X|十六進制|255.ToString("X")|FF|

  
`D`那個就跟數字補0差不多了吧。  
有些還可以自動四捨五入，如`G`。
